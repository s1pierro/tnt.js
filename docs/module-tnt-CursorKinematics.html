Class: CursorKinematics

tnt~CursorKinematics(optsopt)

Positionnement géométrique du curseur déporté. Maintient le curseur à exactement `dist` px du doigt — barre rigide, sans ressort. La direction est conservée : quand le doigt bouge, la barre pivote autour du contact sans changer de longueur. Les coordonnées `x`, `y` de cette classe sont dans le même repère que les coordonnées `touchX`, `touchY` fournies à `update()` (en pratique, relatives à l'élément si on utilise les valeurs émises par TouchEngine).
